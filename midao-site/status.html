Status / future

Current status of the project and it's improvement plan

  • Current Status
  • Future

1. Current status

Current version of Midao JDBC is 0.9.5

Possible improvements which may come with Midao JDBC 0.9.6 are: additional improvements to XML input/output handler (call execution, statement/type handler per query, refresh mechanism in case of file change) and further work on Extended QueryInputHandler which would allow to set more inline configuration for query parameters.

Midao JDBC 0.9.6 is planned to be released in December-January 2013/2014


Change log can be viewed from: GitHub

2. Future

Possible 0.9.2 release

Possible 0.9.3 release

10. (done) Create new Output handlers which would allow to use named queries from external files
